Teacher Help
Change grades
 Select "Grades" from the homepage or from a particular course. Then, select a course, a student and an assignment from the corrseponding dropdown menu (please note the course dropdown is only available when viewing all grades). Enter their grade and press "Submit." If there is already a grade for the grouping, the grade will be updated to the new value entered, otherwise a new grade will be created.

 File repository
 From the homepage, click "File repository." From here, you can see the files that you have access to. Use the search bar to find your desired file.
From the "Actions" column, you can Download a file or Delete it from your personal repo.
Store your personal files with the "Upload File" button or use "Shared With Me" to view files from other students.
